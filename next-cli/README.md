引导的 [Next.js](https://nextjs.org/) 项目。

## 开始

首先，运行开发服务器：

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

用浏览器打开 [http://localhost:3000](http://localhost:3000)，查看结果。

你可以通过修改 `app/page.tsx` 开始编辑页面。编辑文件时，页面会自动更新。

该项目使用
[`next/font`](https://nextjs.org/docs/basic-features/font-optimization)自动优化和加载
Inter（一种自定义谷歌字体）。
